    Getting started
  • Download jquery file from jquery.com

Basic syntax

    $(selector).action
  • The ($) sign is to access jQuery
  • The (selector) refers to a html element,class or id
  • The action refers to a jQuery action.

Document Ready event

before you start writing jQuery you should make a habbit of using this function : $(document).ready(function(){}); and putting your code between it this basicaly uses the code once the page has loaded.

jQuery Selectors

jquery selectors select,handle,find and select HTML elements by using there id,classes,attributes,types,values of the attributes and names examples below :.

    Examples
  • $('p') selects all paragraphs
  • $('#p1' selects all elements with the id p1)
  • $('.heads') selects all elements with the class of heads
  • All selectors in jQuery start with $ sign and are placed in ().
